
파일 입출력: 지금까지 우리가 사용했던 것과 조금 다른 방식의 입출력 방식임

*컴퓨터 같은 경우에는 기본적으로 사용자로 부터 인풋 즉 입력을 밭아서 어떠한
처리를 한뒤에 출력 아웃풋을 내보내주는 그러한 시스템 자체를 의미하잖아요 
지금까지는 이제 우리가 printf,scanf 같은것을 이용해서 실질적으로 사용자가 
텍스트 창에다가 뭘 입력해서 상호작용을 컴퓨터와 할수 있는 형태였다면  

이 파일 입출력이라는건 이제 메모장과 같은 아니면 데이터 파일과 같은 그러한 각종 
파일들을 이용해서 컴퓨터와 상호작용하는것을 의미함 

즉 파일입출력 이라는것은 파일을 다루어서 컴퓨터를 이용하는 방법이라고 할수 있겠음 



파일 입출력은 다양한 프로그램을 개발할때 자주 사용되는 기능입니다. 
( 동빈나 경우는 10년전에 게임보이같은것을 이용해서 포켓몬스터나 다양한 게임같은것을 했었는데 이러한 게임같은경우는 세이브나 불러오기 같은것이 기본적으로 제공이 됩니다. 하지만 
이런 프로그램들은 다 이 클라이언트 즉 우리에 컴퓨터 안에서만 저장이 이루어지는 게임이기 때문에 온라인과 다르게 서버에 저장되는것이 아니라 데이터 모든불러오기 저장하기 같은 데이터가
우리 컴퓨터 안에서 저장이 이루임니다 즉 우리의 컴퓨터 안에 특정한 파일로써 저장이 이루어 지는거죠 그래서 예를들면 옛날에 어떤 cd게임같은 것을 사용하다가 재미가 없으면 cd게임을 뜯어가지고 길행했을때 게임머니 같은것을 조작하거나 이러한 다양한 활동이 성행했었음  )

예를 들면 파일입출력을 통하여 게임에서 현재의 데이터를 저장하고 다시 불러오거나 할 수 있습니다. 뿐만 아니라 프로그램 전반에서 사용되는 다양한 데이터를 처리하고 저장하여 보관할수 있다는 점에서 유용합니다.

또한 이러한 파일 입출력 같은 경우는 보안같은 곳 에서도 되게 많이 활용이 되는데요
그이유를 설명하자면 

c언어 같이 굉장히 강력한 파일 입출력 체계를 가지고 있는 그러한 언어에서는 현재 우리 
컴퓨터에 설치되어 있는 어떠한 파일을 지워라, 복사하여라 이러한 명령또한 가능하기 때문에 
프로그램을 잘못 짜거나 아니면 악의적으로 우리 컴퓨터 내에 설치된 파일을 건드리거나 하는
프로그램을 만들수도 있는것임 

그렇기 때문에 파일입출력은 굉장히 중요한 부분이면서 아주 유용한 부분이기 때문에
양날의 검이라고 할수있음   


기본적으로 c언어에서 제공하는 파일이라는 변수는 

                               create 창조하다(파일을 만들다)
                                         ^
                                     ____|___
      (파일을 열다) 열다 open   <-  | FILE | -> close 닫다(파일을 닫다) 
      (파일을 읽다) 읽다 read   <-  ㅡㅡㅡㅡ -> write 쓰다(파일을 쓰다)// 메모장같은 파일을 하나 새롭게 만들때 그파일안에 글을 쓴다는 의미함 
                                      |  |
      (파일 이름을 다시 짓다) rename <    > delete 삭제(파일을 삭제)

위와같이 다양한 기능들을 제공하고 있음 




복습 
c언어의 파일 입출력에 대해서 한번 알아 보았는데요?

그 프로그램을 실행했을때 그프로그램에 실행되고있는 컴퓨터 즉 운영체제나 어떠한 파일들 폴더들에 쉽게 접근하고 
어떠한 파일을 생성하며 수정할수있는 갖갖이 기능들을 c언어에서 제공하고 있다는것을 알아보았음 

파일 입출력을 통해서 간단하게 텍스트 파일에서 어떠한 정보를 읽거나 쓰거나 그런것들이 가능하다는것을 알아보았었음
