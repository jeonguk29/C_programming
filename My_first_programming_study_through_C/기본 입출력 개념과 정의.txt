기본 입출력 (c언어에 기본적인 입출력 방식을 알아보자)
기본 입출력이란 사용자와 컴퓨터과에 상호작용 그자체를 의미함 흔히말해서 input,output 이라고 할수있음
기본 입출력 같은경우는 우리가 이미 사용하고 있는데 
#include <stdio,h> 스텐다드(기본) 인풋 아웃풋(입출력) 헤더파일(컴퓨터 머리말, 컴퓨터에서 출력될때 각페이지맨윗부분에 자동으로 붙는부분) 이처럼 자동으로 표준 입출력 부분이 모두기능적으로 이헤더파일 라이브러리안에 들어가서 우리가 사용할수 있게 되는것이고 (심지여 계속 공부하면서 모르게 사용하고 있었던것임 )

C언어에서는 기본입출력 함수를 이용하여 사용자와 상호작용을 할수있는데 대표적으로 scanf("%d,&x");는 x라는 정수에 
사용자에게 입력밭은 값을 삽입한다는 의미의 코드 입니다.  
ex 
int main(void)
{
  int x;
  scanf("%d",&x);  // 이렇게 scanf를 넣는다면은 콘솔창에서 사용자가 입력한 첫번째 정수가 이x라는 값에 들어가게 됨
  return 0;            즉 x라는 변수안에 특정한 값(사용자가 지정한)을 삽입할수있는 코드임 
}                    
                  // 문법을 보면 &x 이렇게 &엔퍼센드를 이용하는데 이엔퍼센드가 의미하는것은 특정한 변수에  
                     메모리 주소 자체를 의미하는것임 (나중에 좀더 자세하게 강의로 배울예정)

다만 사용자로 부터 입력밭은 자료형에 따라서 %d, %c, %f, %lf(엘 에프) 등의 다양한 문법을 사용해야한다는것에 
유의해야함 그리고 반면에 프로그램에서 사용자에게 메세지를 출력할때는 가장 대표적으로 printf();를 이용합니다.
(즉 저러한 문법을 사용하고 메세지를 출력할때 가장 대표적인게 프린트라는 뜻임) 
물론 이들은 텍스트 기반(기호의 복합체)의 입출력이며 나중에 GUI를 배우게 되면 사용자에게 글자 뿐만아니라 
이미지 또한 보여줄수있음
